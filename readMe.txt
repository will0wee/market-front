Site de vente en ligne de nourriture,
il est possible (en tant qu'invité), de voir les produits, de créer un panier, de s'inscrire et de se connecter.
Une fois connécter en user, il est possible de commander (en plus de faire les actions de l'invité),
Si connecté en tant que producer, il est possible de gérer ses produits (CRUD), en plus de faire les actions du user
Si connecté en tant qu'admin, il est possible de gérer l'ensemble des produits(CRUD), de gérer l'ensemble des utilisateurs (CRUD), 
de gérer l'ensemble des villes(CRUD) et de gérer l'ensemble des catégories(CRUD), en plus de faire les actions du user
Les commandes sont des données calculés il ne peut donc pas les modifiers.

utilise la librairie inspinia pour le visuel
